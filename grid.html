Grid

WOLASCSS comes with a responsive, full width grid system used for laying out page elements. The grid is split up into rows and columns.

Layout

The main container element for the grid is a div with the class of .g-container. Each row inside the container is a div with the class .g-row. Finally, the columns are split up evenly into 12 columns from .g-col.one to .g-col .twelve. Each column has 20px padding on the right hand side.
