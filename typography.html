Typography

The typography styles for WOLASCSS.

Font

The font used for WOLASCSS is Open Sans. The font files are used with @font-face to deliver them on the web. The files are included in this project. The fallback fonts are in order of preference 'Helvetica Neue', Helvetica, Arial, Sans-Serif.

Body Copy

All body copy should be surrounded by p tags. The font size is set to 14px on the body, and the p tag is set to 100% of that font-size. The default line-height is 1.6em. Paragraphs have a 10px margin-bottom.

Headings

All headings are available, h1 to h6. All headings have a 15px top and bottom margin, and all are bold. The .mid and .right class modifiers can be used to align the headings (you will still need to align anything else yourself).

Paragraphs

Below is a sample of text in a regular p tag:

Pellentesque habitant morbi tristique senectus et netus et malesuada fames ac turpis egestas. Vestibulum tortor quam, feugiat vitae, ultricies eget, tempor sit amet, ante. Donec eu libero sit amet quam egestas semper. Mauris placerat eleifend leo. Quisque sit amet est et sapien ullamcorper pharetra. Vestibulum erat wisi, condimentum sed, ornare sit amet, wisi. Aenean fermentum, elit eget tincidunt condimentum, eros ipsum rutrum orci, sagittis tempus lacus enim ac dui. In turpis pulvinar facilisis. Ut felis.

There are several modifiers avaialable for text inside p tags. For bold text, you can use either b or strong tags.

<p><strong>Pellentesque habitant morbi</strong> tristique senectus et netus et malesuada</p>

Pellentesque habitant morbi tristique senectus et netus et malesuada

For italic text, you can use either i or em tags.

<p><em>Pellentesque habitant morbi</em> tristique senectus et netus et malesuada</p>

Pellentesque habitant morbi tristique senectus et netus et malesuada

To create a paragraph with smaller text, use the small tag. This will make the font 85%.

<p><small>Pellentesque habitant morbi tristique senectus et netus et malesuada</small></p>

Pellentesque habitant morbi tristique senectus et netus et malesuada

Finally, the .intro class can be applied to a paragraph tag to give it a big impact. Use this when introducing a feature etc.

<p class="intro">Pellentesque habitant morbi tristique senectus et netus et malesuada</p>
<p>Then some normal paragraph text under the intro.</p>

Pellentesque habitant morbi tristique senectus et netus et malesuada

Then some normal paragraph text under the intro.

Lists

Both ordered and unordered lists can be used. Both have a margin value of 10px 0 20px 0. Both support one level of sub-listing.

Ordered list
<ol>
	<li>List item 1</li>
	<li>List item 2</li>
	<li>List item 3</li>
	<ol>
		<li>Sub-list item 1</li>
		<li>Sub-list item 2</li>
		<li>Sub-list item 3</li>
	</ol>
</ol>
  1. List item 1
  2. List item 2
  3. List item 3
    1. Sub-list item 1
    2. Sub-list item 2
    3. Sub-list item 3
Unordered list
<ul>
	<li>List item 1</li>
	<li>List item 2</li>
	<li>List item 3</li>
	<ul>
		<li>Sub-list item 1</li>
		<li>Sub-list item 2</li>
		<li>Sub-list item 3</li>
	</ul>
</ul>
  • List item 1
  • List item 2
  • List item 3
    • Sub-list item 1
    • Sub-list item 2
    • Sub-list item 3

Quotes

You can use the blockquote tag to show quotes if required. Probaly won't need to use this very often.

<blockquote>This is a very profound quote.</blockquote>
This is a very profound quote.

Jumbo

You can make a jumbo div and header like the one at the top of this page, using the .jumbo class on a div, which then contains a h1 and h2 tag.

<div class="jumbo">
	<h1>This is the big one</h1>
	<h2>And this is the subheader.</h2>
</div>

This is the big one

And this is the subheader.

Code

You can also use pre and code tags together to format code on the page, or anything else that needs to be displayed in a monospaced font. You must use pre then code inside, and the spacing that is used will determine how the code is formatted.

This:

<pre><code><div class="jumbo">
	<h1>This is the big one</h1>
	<h2>And this is the subheader.</h2>
</div></pre></code>

Will produce this:

<div class="jumbo">
	<h1>This is the big one</h1>
	<h2>And this is the subheader.</h2>
</div>
