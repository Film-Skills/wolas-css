Tables

Different table styles for WOLASCSS.

Table

To avaoid interfering with other table systems, the .w-t class must be used on all <table> tags. Headers should be inside <th> tags. There should be a <tbody> tag just inbetween the <table> tags.

<table class="w-t">
	<tbody>
		<tr>
			<th>#</th>
			<th>Name</th>
			<th>Email</th>
			<th>Phone</th>
		</tr>
		<tr>
			<td>1</td>
			<td>Happy Gilmore</td>
			<td>happy@golfcourse.com</td>
			<td>0400 123 456</td>
		</tr>
	</tbody>
</table>

Table with the lot

There are two modifier classes that can be used for tables. The .striped class adds a zebra-stripe effect to even numbered rows. The .shadow class adds a subltle drop shadow to the table. Both are demonstrated below:

<table class="w-t striped shadow">
	<tbody>
	...
	</tbody>
</table>
