Cards

Cards are a way to split up content. They work together with grids to create a clean and separated interface.

Cards fill the full width of the .g-col element that they occupy. The simplest syntax is:

<div class="c-outer">
  <div class="c-inner">
	<p>
		Content goes here
	</p>
  </div>
</div>

Which will produce the following output, which has a min-height value of 200px:

Content goes here.

The .compact class can be added to .c-outer to create a card that has no minimum height.

<div class="c-outer compact">
  <div class="c-inner">
	<p>
		Content goes here
	</p>
  </div>
</div>

Example:

Content goes here.

Finally, a title can be added to the card by including a div with a .c-title class.

<div class="c-outer">
  <div class="c-title"><p>This is a card title</p></div>
  <div class="c-inner">
	<p>
		Content goes here
	</p>
  </div>
</div>

Which produces this card:

This is a card title

You can also apply alert styles to cards, which will color them accordingly. You can use .a-info, .a-warn, .a-erro, and .a-succ on the .c-outer div.

WOLAS Lite's feed uses a rounded and modified version of the card system. The card-title div must be att the bottom and have the rounded class applied to it. The c-outer div must also have the rounded class applied. The c-inner div must have the nopad class applied because the content will be full width. There is also a secondary c-title with a lite class applied that will show the actual title while the other one will hold hashtags and other info. The code looks like:

<div class="g-col two">
	<div class="c-outer rounded compact">
		<div class="c-inner nopad"><img src="img/car-spray-painting.jpg" /></div>
		<div class="c-title lite">
			<p><strong>Painting a car</strong></p>
		</div>
		<div class="c-title rounded">
			<p><a href="#">#painting</a> <a href="#">#car</a></p>
		</div>
	</div>
</div>
